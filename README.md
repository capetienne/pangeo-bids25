## Tutorial: Earthly marvels revealed: Pangeo, AI, and Copernicus in action 
This tutorial is created in collaboration with the European Pangeo and [Horizon Europe EO4EU projec](https://www.eo4eu.eu) communities for a tutorial session of [2025 Big Data from Space](http://bigdatafromspace2025.org).

This tutorial will provide a comprehensive introduction along with hands-on examples to help you understand how these technologies can be used for Earth science data analysis and interpretation.
